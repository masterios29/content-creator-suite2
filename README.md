# 🎬 Content Creator Suite

**App para crear videos virales para TikTok, Reels y YouTube Shorts con Inteligencia Artificial**

## ✨ Características

### 📦 Módulo 1: Video Clipper
- Sube videos largos y divídelos automáticamente en clips de 45-60 segundos
- Descarga videos desde URLs (YouTube, TikTok, Vimeo, etc.)
- Subtítulos automáticos con IA (Español e Inglés)
- Formato vertical 9:16 listo para redes sociales

### 📦 Módulo 2: Texto a Video
- Convierte historias de Reddit, Wattpad o textos propios en videos
- Voz con IA en Español e Inglés (4 voces por idioma)
- Videos de fondo automáticos (Gaming, Naturaleza, Ciudad, etc.)
- Subtítulos animados estilo TikTok viral
- Música de fondo sin copyright

## 🌐 Idiomas Soportados
- 🇪🇸 **Español** - Interfaz y voces completas
- 🇺🇸 **English** - Full interface and voices

## 🚀 Despliegue en Vercel (Recomendado)

### Paso 1: Crear cuenta en GitHub
1. Ve a [github.com](https://github.com)
2. Crea una cuenta gratuita

### Paso 2: Subir el código a GitHub
1. Crea un nuevo repositorio llamado `content-creator-suite`
2. Sube todos los archivos de esta carpeta

### Paso 3: Desplegar en Vercel
1. Ve a [vercel.com](https://vercel.com)
2. Regístrate con tu cuenta de GitHub
3. Haz clic en "New Project"
4. Importa tu repositorio
5. Haz clic en "Deploy"
6. ¡Listo! Tu app estará en `tu-app.vercel.app`

## 🛡️ Tecnologías

- **Framework**: Next.js 15 con App Router
- **Lenguaje**: TypeScript 5
- **Estilos**: Tailwind CSS 4 + shadcn/ui
- **IA**: z-ai-web-dev-sdk para TTS
- **Base de Datos**: Prisma ORM con SQLite
