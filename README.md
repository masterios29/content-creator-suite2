# 🎬 Content Creator Suite

**App para crear videos virales para TikTok, Reels y YouTube Shorts**

## ✨ Características

- **Video Clipper**: Divide videos largos en clips de 45-60 segundos
- **Texto a Video**: Convierte historias en videos con voz IA
- **Soporte Multiidioma**: Español e Inglés

## 🚀 Despliegue en Vercel

### Paso 1: Subir a GitHub
1. Crea un repositorio en GitHub
2. Sube todos los archivos de este proyecto

### Paso 2: Desplegar en Vercel
1. Ve a vercel.com
2. Importa tu repositorio
3. Click en Deploy
4. ¡Listo!

## 🛠️ Desarrollo Local

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## 📱 Plataformas

- TikTok
- Instagram Reels
- YouTube Shorts
